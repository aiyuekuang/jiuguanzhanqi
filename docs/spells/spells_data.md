# 酒馆战棋法术数据详情

## 发现法术

### 1. 发现随从 (Discover Minion)
- **描述**：发现一个随从
- **费用**：0金币
- **获取方式**：合成金色随从后获得
- **使用效果**：从当前酒馆等级或更高等级的随从中选择一个加入手牌
- **特殊说明**：如果当前酒馆等级为6，则从6级随从中选择

### 2. 发现法术 (Discover Spell)
- **描述**：发现一个法术
- **费用**：0金币
- **获取方式**：通过英雄技能获得
- **使用效果**：从可用法术中选择一个
- **特殊说明**：部分英雄技能可以产生此法术

## 英雄技能法术

### 1. 红龙女王 (Red Dragon Queen)
- **描述**：将你的英雄技能替换为"发现一个随从"

- **获取方式**：阿莱克丝塔萨的英雄技能

### 2. 蛛魔之卵 (Nerubian Egg)
