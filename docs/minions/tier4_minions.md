# 4级随从数据

## 概述
4级随从是酒馆战棋中的中等强度随从，每个随从在池中有15个副本。这些随从通常具有不错的攻击力和生命值，以及一些有用的特殊能力，适合在游戏中期使用。

## 4级酒馆法术

### 1. 防御者的仪式 (Defender's Ritual)
- **法术类型**：酒馆法术
- **法术描述**：使一个随从获得+3生命值和嘲讽
- **法术费用**：2金币
- **获取方式**：鲍勃酒馆
- **使用技巧**：提升生命值并给予嘲讽，保护其他随从

### 2. 自然祝福 (Nature's Blessing)
- **法术类型**：酒馆法术
- **法术描述**：使一个随从获得+2/+2
