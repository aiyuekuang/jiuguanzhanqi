# 2级随从数据

## 概述
2级随从是酒馆战棋中的初中级随从，每个随从在池中有15个副本。这些随从通常具有中等攻击力和生命值，拥有更多样化的特殊能力，是游戏中期的重要组成部分。

## 2级随从列表

## 策略建议

### 早期过渡
2级随从是从1级向更高级别过渡的关键。重点选择具有成长性的随从：
- **商贩元素**：提供经济优势
- **贝类收藏家**：配合鱼人流派
- **爆裂飓风**：配合元素流派

### 中期发展
利用2级随从的特殊能力建立优势：
- **挑衅的船工**：优秀的前排护盾
- **死亡群居蛛魔**：持续成长的攻击力
- **心灵泥魔**：法术流派的核心

### 后期价值
某些2级随从在后期仍有价值：
- **永恒骑士**：提供重生效果
- **耐心的侦查员**：经济操作优化

## 注意事项
- 2级随从池每个有15个副本
- 关注三连合成机会
- 根据英雄技能选择合适的随从
- 考虑流派搭配和后期发展潜力
