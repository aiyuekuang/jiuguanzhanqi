# 1级随从数据

## 概述
1级随从是酒馆战棋中最基础的随从，每个随从在池中有16个副本。这些随从通常具有较低的攻击力和生命值，但有一些有用的特殊能力。

## 1级随从列表

### 1. 雄斑虎 (Alleycat)
- **攻击力**：1
- **生命值**：1
- **种族**：野兽
- **特殊能力**：战吼：召唤一个1/1的雄斑虎
- **获取方式**：鲍勃酒馆
- **金色版本**：2/2，战吼：召唤两个1/1的雄斑虎

### 2. 鱼人猎潮者 (Murloc Tidehunter)
- **攻击力**：2
- **生命值**：1
- **种族**：鱼人
- **特殊能力**：战吼：召唤一个1/1的鱼人斥候
- **获取方式**：鲍勃酒馆
- **金色版本**：4/2，战吼：召唤两个1/1的鱼人斥候

### 3. 微型机器人 (Micro Mummy)
- **攻击力**：1
- **生命值**：2
- **种族**：机械，亡灵
- **特殊能力**：亡语：召唤一个1/1的微型机器人
- **获取方式**：鲍勃酒馆
- **金色版本**：2/4，亡语：召唤两个1/1的微型机器人

### 4. 小恶魔 (Wrath Weaver)
- **攻击力**：1
- **生命值**：3
- **种族**：恶魔
- **特殊能力**：战吼：对你的英雄造成1点伤害，获得+2/+2
- **获取方式**：鲍勃酒馆
- **金色版本**：2/6，战吼：对你的英雄造成1点伤害，获得+4/+4

### 5. 小野猪 (Deck Swabbie)
- **攻击力**：2
- **生命值**：2
- **种族**：海盗
- **特殊能力**：战吼：使你的酒馆升级费用减少1金币
- **获取方式**：鲍勃酒馆
- **金色版本**：4/4，战吼：使你的酒馆升级费用减少2金币

### 6. 小元素 (Sellemental)
- **攻击力**：2
- **生命值**：2
- **种族**：元素

- **获取方式**：鲍勃酒馆
- **金色版本**：4/4，战吼：如果你控制另一个元素，获得+2/+2

### 7. 小海盗 (Scallywag)
- **攻击力**：1
- **生命值**：1
- **种族**：海盗
- **特殊能力**：亡语：召唤一个1/1的海盗，它立即攻击
- **获取方式**：鲍勃酒馆
- **金色版本**：2/2，亡语：召唤两个1/1的海盗，它们立即攻击

### 8. 小野猪人 (Sun-Bacon Relaxer)

- **种族**：野猪人

- **金色版本**：2/4，战吼：获得2个鲜血宝石

- **金色版本**：2/4，战吼：对随机敌方随从造成2点伤害（每有一个友方龙族重复一次）

## 1级随从统计
- **总数量**：16个
- **种族分布**：
  - 野兽：2个
  - 鱼人：1个
  - 机械：2个
  - 恶魔：2个
  - 海盗：3个
  - 元素：2个
  - 野猪人：2个
  - 亡灵：1个
  - 龙族：1个

## 特殊能力统计
- **战吼**：8个随从
- **亡语**：2个随从
- **光环/触发**：2个随从
- **嘲讽**：3个随从
- **无特殊能力**：1个随从

## 注意事项
- 1级随从主要用于游戏初期
- 大部分1级随从都有特殊能力
- 金色版本属性翻倍，特殊能力效果增强
- 随从数据基于当前游戏版本
