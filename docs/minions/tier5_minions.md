# 5级随从数据

## 概述
5级随从是酒馆战棋中的顶级随从，每个随从在池中有9个副本。这些随从通常具有极高的攻击力和生命值，并且拥有非常强大的特殊能力。

## 5级随从列表

### 1. 鱼人领军 (King Bagurgle)
- **攻击力**：6
- **生命值**：3
- **种族**：鱼人
- **特殊能力**：战吼：使其他友方鱼人获得+2/+2
- **获取方式**：鲍勃酒馆
- **金色版本**：12/6，战吼：使其他友方鱼人获得+4/+4

### 2. 机械幼龙 (Sneed's Old Shredder)
- **攻击力**：5
- **生命值**：7
- **种族**：机械
- **特殊能力**：亡语：召唤一个传说随从
- **获取方式**：鲍勃酒馆
- **金色版本**：10/14，亡语：召唤两个传说随从

### 3. 小恶魔 (Mal'Ganis)
- **攻击力**：9
- **生命值**：5
- **种族**：恶魔
- **特殊能力**：光环：其他友方恶魔获得+2/+2，你的英雄免疫
- **获取方式**：鲍勃酒馆
- **金色版本**：18/10，光环：其他友方恶魔获得+4/+4，你的英雄免疫

### 4. 小野猪 (Goldrinn, the Great Wolf)
- **攻击力**：4
- **生命值**：4
- **种族**：野兽
- **特殊能力**：亡语：使你的野兽获得+4/+4
- **获取方式**：鲍勃酒馆
- **金色版本**：8/8，亡语：使你的野兽获得+8/+8

### 5. 小元素 (Lil' Rag)
- **攻击力**：6
- **生命值**：6
- **种族**：元素
- **特殊能力**：战吼：使一个友方随从获得等同于你的酒馆等级的属性值
- **获取方式**：鲍勃酒馆
- **金色版本**：12/12，战吼：使一个友方随从获得等同于你的酒馆等级的属性值

### 6. 小海盗 (Captain Eudora)
- **攻击力**：5
- **生命值**：4
- **种族**：海盗
- **特殊能力**：战吼：随机使一个友方随从获得+1/+1，重复4次
- **获取方式**：鲍勃酒馆
- **金色版本**：10/8，战吼：随机使一个友方随从获得+2/+2，重复8次

### 7. 小野猪人 (Agamaggan, the Great Boar)
- **攻击力**：6
- **生命值**：6
- **种族**：野猪人
- **特殊能力**：光环：你的鲜血宝石额外提供+1/+1
- **获取方式**：鲍勃酒馆
- **金色版本**：12/12，光环：你的鲜血宝石额外提供+2/+2

### 8. 小亡灵 (Ghastcoiler)
- **攻击力**：7
- **生命值**：7
- **种族**：亡灵
- **特殊能力**：亡语：召唤两个亡灵随从
- **获取方式**：鲍勃酒馆
- **金色版本**：14/14，亡语：召唤四个亡灵随从

### 9. 小龙 (Kalecgos, Arcane Aspect)
- **攻击力**：4
- **生命值**：12
- **种族**：龙族
- **特殊能力**：战吼：如果你控制另一个龙族，发现一个法术
- **获取方式**：鲍勃酒馆
- **金色版本**：8/24，战吼：如果你控制另一个龙族，发现两个法术

- 5级随从是游戏后期的核心选择

- 金色版本属性翻倍，特殊能力效果增强
- 随从数据基于当前游戏版本
